# moe-vim-mode

Vim mode for the Monaco Editor.

## Status

Still under heavy development and not published to npm yet.

For list of supported commands, see [status.md](status.md).

## Development setup

``` bash
npm install

# watch and serve demo at http://localhost:8080/
npm run start

# run all tests
npm run test

# build dist
npm run build
```
